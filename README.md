# Node.js CI Project

![CI Pipeline](https://github.com/CristinaLopez796/nodejs-ci-project/actions/workflows/ci.yml/badge.svg)

Proyecto Node.js con integración continua usando GitHub Actions, Jest y ESLint.

## Características

- 3 funciones independientes
- Pruebas unitarias con Jest
- Cobertura mínima del 80%
- Linting con ESLint
- Pipeline CI con GitHub Actions
- Matrix con Node.js 18 y 20
- Caché de dependencias con npm

## Instalación

```bash
git clone https://github.com/CristinaLopez796/nodejs-ci-project.git
cd nodejs-ci-project
npm install
